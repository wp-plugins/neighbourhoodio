Canadian Neighbourhood Demographics

== Description ==

[neighbourhood.io](http://neighbourhood.io) is a WordPress plugin for displaying beautiful chart of of Canadian Neighbourhoods Demographics using geographic coordinates.

The plugin uses Google Visualization API to render charts, which support cross-browser compatibility.

== Documentation ==

neighbourhood.io uses Shortcode syntax to display charts.

`[neigbourhoodio-chart lat="49.1231231" lon="-79.2664966"]`

**Parameters**

* lat: Latitude component of location.
* lon: Longitude component of location

== Installation ==

1. Install SoldPress either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Click the Settings -> neighbourhood.io Add your subscription key.
3. Click Save
